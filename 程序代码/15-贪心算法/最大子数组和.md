    1、最大子数组和（连续）
    【思路】
    初始化一个数组
    每次从数组中往后找一个元素，如果
    
    如果【该值】加上 dp【i】中的前一个值大于【原数组】中的值，那么就更新这个dp【i】
    否则，就把nums[i]这个值给dp

      class Solution:
          def maxSubArray(self, nums: List[int]) -> int:


              #动态规划
              lenght = len(nums)
              dp = [0 for i in range(lenght)]
              dp[0]=nums[0]
              for i in range(1,lenght):
                  value = nums[i]+dp[i-1]
                  if value>nums[i]:
                      dp[i] = value
                  else:
                      dp[i] = nums[i]
              return max(dp)
